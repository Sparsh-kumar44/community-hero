# 🌍 Community Hero

> **Community Hero** is an AI-powered civic engagement platform that enables citizens to report community issues, coordinate volunteers, and collaboratively resolve neighborhood problems using **Google Gemini**, **Firebase**, **Google Maps**, and **Google Cloud Run**.

🚀 **Built for Vibe2Ship 2026 — India's Biggest Vibe Coding Hackathon by Coding Ninjas × Google for Developers.**

## 🎯 Problem Statement

Develop an AI-powered platform that enables communities to efficiently report, prioritize, and resolve civic issues through intelligent automation and collaborative volunteer engagement.

---

## ✨ Highlights

- 🤖 Google Gemini powered issue analysis
- 🖼️ AI-generated image captions
- 🔐 Firebase Google Authentication
- 📍 Google Maps integration
- 👥 Volunteer collaboration
- 🏆 Reputation & leaderboard system
- ✅ Community verification workflow
- ☁️ Deployed on Google Cloud Run
- 🐳 Docker containerized
